Зальешь данный файл (только сначало распакуй) на сервер (желательно Ubuntu) в папку root, далее заходишь на сервер через ssh и устанавливаешь python (если его нет)

После этого, ставить pip командой: apt install python3-pip

После этого, идём в папку с ботом cd папка_с_ботом

После этого устанавливаем библиотеки:
pip install aiogram
pip install scheduler
pip install requests

И запускаем
python main.py




Проверка статистики бота
/stats


Рассылка всем пользователям бота
/all Текст
